dat.GUI
===

A lightweight graphical user interface for changing variables in JavaScript.

Various types of controls
---

#### 1.Number
```
var controls = new function () {
   this.rotationSpeed = 0.02;
};

var gui = new dat.GUI();
gui.add(controls, 'rotationSpeed', 0, 0.5);

```
